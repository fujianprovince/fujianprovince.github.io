这天上了高中第八节社团课，我们做了铜镜实验。其本质是通过让铜盐还原成单质铜镀在盛装其溶液的瓶的壁上，形成一层闪亮的铜镜。
以下是铜镜实验的配方：

  • 称量2g硫酸铜和1g维生素C（C6H8O6），溶解于10mL水中，溶液变绿；
  • 往溶液中滴加10%氨水，此时应持续搅拌，溶液将变墨绿色、再变黄褐色，直至变为无沉淀的绛蓝色溶液时停止滴加；
  • 迅速装瓶，过程中溶液可能变为黄色，装瓶应装满，盖瓶不留气泡；
  • 将瓶子尽量保持静止地倒置约8小时，则镀镜完成。

还有一些注意事项：

  • 溶质完全溶解需要非常充分的搅拌；
  • 全程溶液现配现用，一直保持充分搅拌状态，若静置可能出现亚铜或单质铜沉淀，影响成镜；
  • 维生素C不宜过量，但应大于0.8g；
  • 氨水滴加量在10mL左右，注意要逐滴加；
  • 装瓶宜用胶头滴管，以避免气泡的产生，气泡会使之后单质铜难以镀上瓶底。

实验涉及方程式如下：

  • Cu2+ + C6H8O6 → Cu+ + C6H6O6
  • Cu+ + 2NH3·H2O = [Cu(NH3)2]+ + H2O
  • 4[Cu(NH3)2]+ + O2 + 8NH3·H2O = 4[Cu(NH3)4]2+ + 4OH- + 6H2O
  • 2Cu+ → Cu2+ Cu↓
